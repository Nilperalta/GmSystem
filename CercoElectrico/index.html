Preguntas Frecuentes

¿Con qué sistemas de seguridad cuentan las alarmas?

Actualmente, las alarmas más avanzadas cuentan con 3 sistemas de seguridad:

  • Seguridad ante intrusión: Se activan los detectores de movimiento y se envía una señal por vía GPRS a la Central Receptora de Alarma.
  • Seguridad ante inhibición: Sistema anti-jamming que detecta interferencias en la señal.
  • Seguridad ante sabotaje: Actúa la batería externa en caso de corte de electricidad para que la señal IP siga funcionando con normalidad.

Componentes

Elementos de una Alarma Antirrobo

Los elementos de seguridad más comunes en las alarmas antirrobos son:

Panel Central

El dispositivo principal del sistema que coordina todos los sensores y envía las alertas.

Sensores Magnéticos

Para puertas o ventanas. Detectan cuando se abre un acceso protegido.

Sensores de Movimiento

Interiores o exteriores. Detectan presencia y movimiento en áreas protegidas.

Cámaras de Video Vigilancia

Cableadas o inalámbricas para monitoreo visual en tiempo real.

Sirena de Alta Potencia

De altos decibeles para alertar de inmediato ante una intrusión.

Placa o Sticker

Para fachadas, con efecto disuasorio para posibles intrusos.
